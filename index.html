Quelles données nous envoyer ? (Guides détaillés fournis)

Pas de panique ! Après paiement, vous recevez un guide détaillé pour chaque export + support par email si besoin.

Google Ads

  • Fichier : GoogleAds_90jours.csv
  • Où : Google Ads → Rapports → Campagnes → Exporter CSV
  • Colonnes : Date, Campagne, Impressions, Clics, Coût
  • Période : 90 derniers jours
  • Guide détaillé envoyé après paiement

Shopify/WooCommerce

  • Fichier : Commandes_90jours.csv
  • Où : Admin → Commandes → Exporter → CSV complet
  • Données : Commandes, montants, sources traffic (UTM si possible)
  • Période : 90 derniers jours
  • Instructions pas-à-pas fournies

Facebook Ads

  • Fichier : FacebookAds_90jours.csv
  • Où : Ads Manager → Rapports → Campagnes → Exporter
  • Colonnes : Date, Campagne, Impressions, Clics, Dépense
  • Période : 90 derniers jours
  • Guide d'export détaillé inclus
